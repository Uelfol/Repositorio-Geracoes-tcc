# Repositório de Gerações de IA - TCC

Este repositório serve como apêndice ao Trabalho de Conclusão de Curso (TCC), catalogando todos os assets e testes gerados com o uso de Inteligência Artificial para o design e desenvolvimento de jogos.

## 📂 Estrutura do Repositório

Os arquivos foram organizados de acordo com a metodologia e as etapas desenvolvidas durante a pesquisa:

- **[01_Download_dos_Jogos_Finalizados](./01_Download_dos_Jogos_Finalizados/)**: Contém as builds finais ou links para o download dos jogos desenvolvidos ao longo do projeto.
- **[02_Resultados_Etapa_2_Pesquisa_Exploratoria_Ferramentas](./02_Resultados_Etapa_2_Pesquisa_Exploratoria_Ferramentas/)**: Geração de imagens, textos e áudios resultantes da etapa inicial de testes das ferramentas de IA.
- **[03_Resultados_Etapa_3_Processo_de_Ideacao_e_Selecao](./03_Resultados_Etapa_3_Processo_de_Ideacao_e_Selecao/)**: Resultados da fase de ideação, rascunhos e seleção de ideias usando as IAs.
- **[04_Resultados_Etapa_4_Primeiro_Estudo_de_caso_PinkBob](./04_Resultados_Etapa_4_Primeiro_Estudo_de_caso_PinkBob/)**: Assets gerados para o primeiro estudo de caso, "PinkBob J. e o Enigma do Rum", divididos por episódios e tipos de assets (Cenários, Músicas, Personagens, SFX).
- **[05_Resultados_Etapa_5_Segundo_Estudo_de_caso_Hacking_Survivors](./05_Resultados_Etapa_5_Segundo_Estudo_de_caso_Hacking_Survivors/)**: Assets gerados para o segundo estudo de caso, "Hacking Survivors".
- **[06_Resultados_Etapa_6_Terceiro_Estudo_de_caso_Pafuncio](./06_Resultados_Etapa_6_Terceiro_Estudo_de_caso_Pafuncio/)**: Assets gerados para o terceiro estudo de caso, "Pafúncio Vs Desenhos".

## 📖 Como navegar

Cada subpasta e divisão de estudo de caso deste repositório contém o seu próprio arquivo `README.md`. 
Graças à estrutura do GitHub, ao navegar por qualquer pasta, o `README.md` será aberto automaticamente exibindo um **catálogo completo** das gerações daquela etapa.

Nestes catálogos internos, você encontrará:
- O nome e o arquivo gerado (com preview da imagem, se aplicável).
- O modelo de IA utilizado (ex: Midjourney, ChatGPT, Suno, etc.).
- O prompt exato enviado para a IA.
- A saída/resultado da IA (para os casos de diálogos ou textos).
- Observações adicionais sobre cada resultado e escolhas do processo.
